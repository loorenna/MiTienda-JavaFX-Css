# MiTienda

## Descripción

MiTienda es una aplicación desarrollada en JavaFX que simula un sistema básico de gestión de productos. El sistema cuenta con un inicio de sesión funcional, un menú lateral de navegación y un módulo para administrar productos mediante operaciones CRUD.

## Tecnologías Utilizadas

- Java
- JavaFX
- CSS
- Scene Builder
- GitHub

## Funcionalidades Implementadas

- Inicio de sesión con cualquier usuario, contraseña y rol de administrador.
- Menú lateral de navegación.
- Registro de productos.
- Actualización de productos.
- Eliminación de productos.
- Limpieza de formularios.
- Visualización de productos en una tabla.
- Aplicación de estilos CSS.

## Instrucciones para Ejecutar

1. Clonar el repositorio desde GitHub.
2. Abrir el proyecto en IntelliJ IDEA o NetBeans.
3. Configurar JavaFX en el proyecto.
4. Ejecutar la clase Main.java.
5. Iniciar sesión para acceder al dashboard.
